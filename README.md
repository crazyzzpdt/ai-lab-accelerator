## 项目简介

如果你在晶泰科技智能自主实验室担任研发效能负责人，你会如何基于飞书会议 AI 能力，打造一套「AI 实验研发加速器」？

本方案打通 **方案研讨 → 参数评审 → 实验执行 → 结果复盘 → 迭代优化 → 知识沉淀** 全流程，让一次实验的关键结论能在 24 小时内被全团队检索、理解和复用。
